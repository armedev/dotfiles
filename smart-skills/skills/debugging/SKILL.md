## Debugging

Use a systematic approach to find and fix bugs efficiently.

* Reproduce the bug consistently before attempting fixes
* Isolate the problem - determine the minimal case that triggers it
* Use logging, print statements, or a debugger to trace execution
* Form a hypothesis about the root cause before making changes
* Change only one thing at a time and observe the result
* Check recent changes in version control for clues
* Verify assumptions about input data and state
* Test the fix thoroughly - ensure it works and doesn't break other things
* Document the bug and solution for future reference
